Zusammenfassung

Die Applikation dient der Erstellung von Qualitätsregelkarten für kleine Labore.

Der User kann sich für jeden Parameter der im Labor gemessen wird, eine neue Qualitätsregelkarte erstellen. Darauf hin hat der User die Möglichkeit Messwerte einzutragen. Ab 2 Messwerten, wird automatisch vom System eine Qualitätsregelkarte erstellt die dann auch in der App angezeigt wird. So hat der Qualtitätsbeauftragte immer den Überblick ob Geräte neu kalibriert werden müssen und alle Parameter im Sollbereich liegen.
